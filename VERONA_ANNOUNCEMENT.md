# Verona Announcement

## Slack or Teams post

We now have a small internal plugin called `Verona` for `.vomd`, `.vocs`, and `.voud` files.
Those files are JSON in practice, and Verona makes them open and format properly in VS Code and JetBrains.

Install links:
- Release page: `https://github.com/iqb-berlin/verona-ide-support/releases`
- VS Code: release asset `verona-vscode-1.0.1.vsix`
- JetBrains: release asset `verona-jetbrains-1.0.1.jar`

Main benefit:
- open Verona files without editor confusion
- format them as JSON quickly
- reduce review and debugging friction

## Short email version

Subject: Verona plugin for `.vomd`, `.vocs`, and `.voud`

We now have an internal plugin called `Verona` that improves support for `.vomd`, `.vocs`, and `.voud` files in both VS Code and JetBrains.
Because those files are effectively JSON, Verona makes them much easier to open, inspect, and format correctly.

If you work with these files regularly, please install it from the packaged artifacts in this repository.
